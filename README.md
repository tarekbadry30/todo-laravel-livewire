## Installation

- rename the ".env.example" to be ".env"


- install composer packages

```bash
  composer install 
```

- install npm  packages

```bash
  npm install 
```
- configure your database setting 
```bash
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=corseat
DB_USERNAME=root
DB_PASSWORD=
```
- migrate database tables

```bash
  php artisan migrate 
```

- run default Seeder

```bash
  php artisan db:seed 
```
    
- run project

```bash
  php artisan serv
```
